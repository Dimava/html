# html

Static GitHub Pages site for `https://dimava.github.io/html/`.

## Publish

1. Create the GitHub repository `html` under the `Dimava` account.
2. Add this folder as the repo remote.
3. Push `main`.
4. In GitHub, open `Settings -> Pages`.
5. Set `Build and deployment` to `Deploy from a branch`.
6. Select branch `main` and folder `/ (root)`.

After GitHub Pages finishes, the site URL will be:

`https://dimava.github.io/html/`
